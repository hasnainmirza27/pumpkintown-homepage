How to Connect      (2018-02-11)

In order to connect to the server you need to be using the Java version of Minecraft. This means no XBOX or Windows 10 versions. If you've always been using the same one for years, it's probably the Java one, no worries.
The address is simply "pumpkintown.duckdns.org", no port needed!

Server Mods      (2018-02-11)

I like to have a Vanilla+ style of experience and so the server feels fairly plain but there are a few server mods running for quality of life improvements.

  • Multiverse -- Allows us to have a Creative Mode world in the town castle basement. At the far end on the right!
  • Lockette -- By attaching a sign to a chest (or other blocks) you can lock it. You can add people who can access it, too.
  • SetHome -- You can stand at your home and type "/sethome" to set a warp point that "/home" instantaneously warps to. No limits, but try to be fair about fleeing!
  • EasyWarp -- You can set arbitrarily named warp points and warp to them. 10 second warm up before warp. Try /help ezwarp
  • DynMap -- It's over there! To the left! It shows the server.
